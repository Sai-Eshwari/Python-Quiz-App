# Python Quiz App

This is a simple interactive quiz application I built while learning Python.  
The goal of this project was to understand core Python concepts and get hands-on experience with building a GUI using Tkinter.

---

##  About the Project

The application presents multiple-choice questions, evaluates the user's answers, and calculates the final score.  
It also stores results for later review.

This project helped me move from basic Python to building a working application.

---

##  Features

- Interactive quiz interface using Tkinter
- Multiple-choice questions
- Score calculation at the end
- Stores results in a CSV file
- Beginner-friendly design

---

##  Tech Stack

- Python
- Tkinter (for GUI)
